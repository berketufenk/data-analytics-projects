## 📐 Analytics Methods

- **Cohort Analysis** — retention curves segmented by acquisition period
- **A/B Testing** — hypothesis testing, statistical significance, sample sizing
- **Funnel Analysis** — step-by-step conversion tracking and drop-off analysis
- **RFM Segmentation** — Recency, Frequency, Monetary user segmentation
- **North Star Metric** — defining and tracking the single metric that matters
